It is a website that brings all the online pages of Bangladesh under one platform. Through this website customers can easily find their desired page as the pages are categorized here. As the pages are categorized here this website can be time efficient for the customers.

Users Are Categorized By:
  • Admin
  • Page Owner
  • Customer

  • Admin Features
    • Admin has accessed to all the features of page owners and users. Can edit,delete & modify anything.
  • Page Owners Features
    • Page owner has edit access only to his/her own pages.
    • Page owner can add their product information,cateogories of product they sell, contacts etc.
    • Page owner can see how many viewer have visited their page.
